Simple Renew
==============

Simple Renew manages renewable subscriptions to your site.

## Features

* Integrated with the Recurly API <a href="https://recurly.com/">recurly.com</a>

## Requirements

* PHP 5.3+ with libcurl and openSSL
* Joomla! 2.5 - 3.x
